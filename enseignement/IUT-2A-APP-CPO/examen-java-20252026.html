Exercice 2 – Communication réseau et concurrence (8 points)

Serveur TCP multi-clients

Créer une classe MessageServer qui :

  • Écoute sur le port 5000
  • Accepte plusieurs connexions clientes
  • Crée une classe ClientHandler implémentant l'interface Runnable
  • Délègue chaque client à un thread distinct
  • Le thread lit une ligne envoyée par le client
  • Le thread utilise MessageParser pour générer la réponse
  • Le thread envoie la réponse au client
  • Ferme correctement les ressources

Client TCP

Créer une classe MessageClient qui :

  • Se connecte au serveur sur localhost et le port 5000
  • Envoie un message respectant (ou non) le format imposé
  • Lit la réponse envoyée par le serveur
  • Affiche la réponse dans la console
  • Ferme correctement la connexion
Le message peut être :
  • codé en dur
  • ou passé en paramètre du constructeur du client
